  1. 新增随堂测功能。
    获取随堂测
    xxxxxxxxxx
    DWLive.getPracticeInfomation(pid,callback)
    参数：pid --->随堂测id（没有随堂测则可以不传）
         callback ---->回调参数
    回调函数返回参数：
      无随堂测：

      有随堂测：
      {

                      …
            ]

            status: 1,                //随堂测状态 1 是正在发布 2 结束答题进入统计

          }
        },
        errorCode: 0,
        msg: "操作成功",
        success: true
      }

使用说明：

xxxxxxxxxx
   //直播
   <script type="text/javascript">
    DWLive.init({
      userid: 'userid',//必须参数
      roomid: 'roomid',//必须参数
      groupid: "groupid", //可选
      viewername: 'name',//可选
      viewertoken: 'password',//如果直播间设置为密码验证，必选
      viewercustomua: 'android',//可选
      fastMode:true//可选参数，默认为true，若用户不传该参数则默认为true。（修改序号1）
    });
   </script>
   //回放
   <script type="text/javascript">
    $.DW.config({
          userId: 'userId',//必选参数
          roomId: 'roomId',//必选参数
          recordId: 'recordId',//调整必填项目，若不传会抛出异常。 （修改序号3）
          groupId : "groupId",//可选参数    
          viewername: 'name',//可选参数，若为密码验证则为必选参数
          viewertoken: 'password',//可选参数，若为密码验证则为必选参数
          isH5play:true,//可选参数，该参数调整为默认值为true，如若不传或则不写则为true （修改序号2）
          fastMode:true//可选参数，该参数调整默认值为true，如若不传或则不写则为true （修改序号1）
      });
   </script>
   *注：必须传入recordId才能正常播放，对于之前可以通过传入liveId可播放不再进行支持。
